# SAP Import

Automação de processos SAP GUI desenvolvida em Python, com suporte a múltiplas transações.

## Funcionalidades

* Execução automatizada de diferentes transações SAP
* Aplicação de variantes e filtros
* Seleção dinâmica de parâmetros (centros, datas, etc.)
* Exportação de dados para arquivos
* Interface gráfica com PySide6
* Estrutura modular para fácil expansão

## Tecnologias

* Python
* SAP GUI Scripting
* PySide6

## Objetivo

Centralizar e automatizar rotinas operacionais no SAP, reduzindo esforço manual, erros e tempo de execução.

## Diferencial

Projeto estruturado de forma modular, permitindo a inclusão de novas transações sem alterar a base principal do sistema.
